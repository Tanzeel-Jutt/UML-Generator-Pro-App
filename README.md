# 🎯 UML Generator Pro

**Advanced Diagram Generator for Java, C++, Python & SQL**

> Generate professional UML Class Diagrams, ER/EERD Diagrams, Relational Mappings, SQL Normalization, and SOLID Analysis — all in one tool.

---

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 📊 **UML Class Diagrams** | Full support for classes, interfaces, enums, records with relationships |
| 🔷 **ER Diagrams** | Entity-Relationship diagrams from SQL DDL scripts |
| 🔶 **EERD Diagrams** | Enhanced ER with specialization, multi-valued & derived attributes |
| 📋 **Relational Mapping** | Automatic relational schema generation |
| 📜 **SQL Normalization** | 1NF through 5NF normalization engine |
| 🛡️ **SOLID Analysis** | Detect SOLID principle violations in your code |
| 🎨 **Multiple Themes** | Dark, Light, and Matrix Hacker themes |
| 📤 **Export** | Export diagrams as PNG images |

## 🌐 Use Online (No Download Required)

**👉 [Open UML Generator Pro Web](https://uml-generator-pro.netlify.app/)**

Works directly in your browser — no installation needed!

## 💻 Download Desktop App

Download the latest version from the **[Releases](https://github.com/Tanzeel-Jutt/UML-Generator-Pro-App/releases)** page.

### Requirements
- **Java 21** or higher ([Download Java](https://adoptium.net/))

### How to Run
```bash
java -jar UML-Generator-Pro-1.0.0.jar
```

Or simply **double-click** the JAR file (if Java is properly installed).

## 📸 Supported Languages

- ☕ **Java** — Full AST-based parsing
- 🐍 **Python** — Class and method extraction  
- ➕ **C++** — Header and source file parsing
- 🗄️ **SQL** — DDL CREATE TABLE parsing

## 📝 How to Use

1. **Paste your code** into the editor
2. **Select the language** (Java, Python, C++, or SQL)
3. **Click Generate** — your diagram appears instantly!
4. **Click any class** to see detailed view with relationships
5. **Export** your diagram as PNG
